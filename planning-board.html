1. Map As Board

Use OSM tiles visually, with generated territory polygons overlaid on top. Real features drive economy, defense, mobility, and strategic value.

2. Dynamic Conflict

Start with visible army movement. Road, rail, air, and sea routes all use ETAs; faster infrastructure lowers travel time but never makes movement instant.

3. Humans + NPCs

Humans and NPCs all start small, grow by capture, and can permanently eliminate each other. Humans choose any start; NPC starts are evenly spread.

4. Territory Stats

Factories and shops raise economy. Castles, military sites, and government buildings raise defense. Roads, rail, airports, and ports raise mobility.

5. Map Control Leaderboard

Rank factions by percentage of total map area controlled. The primary win condition is 100% control. Eliminated players are permanent knockouts.
